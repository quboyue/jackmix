Contstructor Build a dialoge with the given title and label, and with the initial values supplied as a list of integers. Midi supports control channels in the range 0-119; 120 and above are reserved for "Channel Mode Messages" (see http://www.midi.org/techspecs/midimessages.php). If the pointer supplied in init is not null, it is assumed to reference an array of int large enough to hold one initial value for each requested conotrol channel.

The controls parameter is a list of strings which identify to the user the slider within the element being controlled, so a Mono-to-Stereo element may request a MIDI channel to control "Pan" and "Gain", for example.

参数
titleDialogue title
labelString (not a QLabel!) to appear at the top of the dialog.
controlsA list of strings identifying the channels.
initInitial values for the control channel numbers in the range 0-119 inclusive.
