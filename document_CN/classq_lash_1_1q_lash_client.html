qLash::qLashClient::qLashClient ( QString  clientname,
int  argc = 0,
char **  argv = 0,
QObject *  p = 0 
)

Construct a LASH-connection.

LASH needs a client-name and needs at least the first of the commandline arguments to determine the executable ( the others will be used to determine session and co if lashd has to autostart... ). If argc and argv both are 0, the arguments will be extracted from QCoreApplication::arguments().

Parameters
clientnameThe name of this client. This should always be the same to restore configs.
argcNumber of arguments stored in argv
argvCommandline arguments.
pThe parent QObject.
